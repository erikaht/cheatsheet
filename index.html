Erika's Cheatsheet

HTML elements

Element Description
<html> The root element for any HTML document, defines the whole document.
<head> Container for meta data - goes after HTML and before Body.
<meta> Provides structured meta data about a page. Self closing.
<link> Specifies relationship between current document and an external resource, used for connecting a CSS Stylesheet to the HTML document for example.
<body> Where the main information for the page goes; the stuff you want people to see on the webpage.
<p> Creates a paragraph.
<br> Creates a line break in the content. Self closing.
<span> Use this when you want to target a specific piece of content within a line or within a paragraph.
<div> Divides pages into sections - useful for grouping elements.

CSS properties

Property Description
background-color Changes the color of the background of the indicated element.
color Changes the font color.
border Specifies the type, color and thickness of the border.
opacity Sets the transparency of the element.
visibility Sets the visibility of an element.
padding Sets the space between the content and the element border.
align-content Aligns items in a flex container along flex lines.
background-image Specifies a background image for an element.
font-family Sets the font family for an element.
